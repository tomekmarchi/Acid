Array#intersect

Returns an new array that is the set intersection of the array and the input array(s).

Array#intersect(array: any, args: ...any, null: ...Array): Array
Parameters
array (any)
args (...any)
null (...Array) *arrays - A variable number of arrays.
Returns
Array: The new array of unique values shared by all of the arrays.
Example
$.intersect([1, 2, 3], [2, 3, 4]);
// -> [2, 3]

$.intersect([1, 2, 3], [101, 2, 50, 1], [2, 1]);
// -> [1, 2]

Array#remove

Removes all occurrences of the passed in items from the array and returns the array.

Note: Unlike `.without()`, this method mutates the array.

Array#remove(array: any, removeTheseArg: any, null: ...any): Array
Parameters
array (any)
removeTheseArg (any)
null (...any) *items - Items to remove from the array.
Returns
Array: The array this method was called on.
Example
var array = [1, 2, 3, 3, 4, 3, 5];

remove(array,1);
// -> [2, 3, 3, 4, 3, 5]

remove(array,3);
// -> [2, 4, 5]

remove(array,[2, 5]);
// -> [4]
